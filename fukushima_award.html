⚠️ データベースの権限設定が必要です

「権限がない（permission denied）」状態です。Firestore のルールを下記に設定すると動きます。

  1. このリンクでルール画面を開く
  2. 表示中のテキストをすべて消す
  3. 下のルールをまるごと貼り付け
  4. 右上の「公開」ボタンを押す（最大1分で反映）
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
